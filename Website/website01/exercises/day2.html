Exercise 1.1

Make a matlab script that takes the errors from a clockwise experiment Xcw and Ycw and the errors from a counter clockwise experiment Xccw and Yccw and the side length L and the nominal width as inputs and calculate Ed and Eb as output. The calculations should be based on the formulas 6.22 to 6.29. andshould be calculated as the mean of the values based on the x-coordinates and the ycoordinates. Test the script with (Xcw, Ycw, Xccw, Yccw, L, b0)=(0.20, 0.20, 0.12, -0.12, 1 ,0.26) The result should be (Ed, Eb)=(0.9948, 0.9515)

%% 3.1 Matlab script to calculate Eb and Ed.
clear
clc
Xcw = -0.05;
Ycw = -0.05;
Xccw = -0.05;
Yccw = 0.05;
L = 3;
b0 = 0.26;

% The calculation of Eb
alphaX = ((Xcw + Xccw) / (-4*L)) * (180/pi);
alphaY = ((Ycw - Yccw) / (-4*L)) * (180/pi);
alpha = (alphaX + alphaY) / 2;

Eb = 90/(90-alpha)

% The calculation of Ed
betaX = ((Xcw - Xccw) / (-4*L)) * (180/pi);
betaY = ((Ycw + Yccw) / (-4*L)) * (180/pi);
beta = (betaX + betaY) / 2;

R = (L/2) / sin(degtorad(beta/2));

Ed = (R + b0/2) / (R - b0/2)

res = Eb * 0.26
